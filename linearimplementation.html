Linear Search

Implementation


Linear search is implemented using following steps...

Consider the following list of elements and the element to be searched...

                             

Step 1 : Read/accept the search element (Key element) from the user.

                            Given key element = 3.


Step 2 : Compare the key element with the first element in the list.

    In this the key element(3) is compared with first element(5)

                             


Step 3 : If both are matched, then display "Element found!!!" and terminate the function.


Step 4 : If both not matched, then compare key element with the next element in the list.

     Both elements did not matched. So move to next element.


Step 5 : Repeat steps 3 and 4 until key element is compared with last element in the list.

     Now, the key element(3) is compared with second element(4)

                             
     Both elements did not matched. So move to next element.

     Now again, the key element(3) is compared with third element(3)

                            
     Both the elements matched. So, we display the element found at index 2 i.e., third element.


Step 6 : If last element in the list also doesn't match, then display "Element not found!!!" and terminate the function.
