Floyd Warshall

Overview


Floyd Warshall Algorithm is a famous algorithm. It is used to solve All Pairs Shortest Path Problem. It computes the shortest path between every pair of vertices of the given graph. Floyd Warshall Algorithm is an example of dynamic programming approach.


Algorithm


        
        Data: A directed weighted graph G(V, E)
        Result: Shortest path between each pair of vertices in G
          for each d € V do
            distance/d]/d] — 0;
          end
          for each edge (s, p) € E do
             distance[s][p] — weight(s, p);
          end
          
        n = cardinality(V);
          for k = 1 ton do
            for 1 = 1 ton do
              for j = 1 ton do
                if distance/i/[j] > distance/i][k] + distance/k]/j] then
                  distance/i]/j] — distance[i][k] + distance/[k]/3/;
                end
              end
            end
          end

Implementation


We’re taking a directed weighted graph G(V, E) as an input. And first, we construct a graph matrix from the given graph. This matrix includes the edge weights in the graph. Next, we insert 0 in the diagonal positions in the matrix. The rest of the positions are filled with the respective edge weights from the input graph.
Then, we need to find the distance between two vertices. While finding the distance, we also check if there’s any intermediate vertex between two picked vertices. If there exists an intermediate vertex then we check the distance between the selected pair of vertices which goes through this intermediate vertex.
If this distance when traversing through the intermediate vertex is less then the distance between two picked vertices without going through the intermediate vertex, we update the shortest distance value in the matrix.
The number of iterations is equal to the cardinality of the vertex set. The algorithm returns the shortest distance from each vertex to another in the given graph.

Time Complexity


  1. Floyd Warshall Algorithm consists of three loops over all the nodes.
  2. The inner most loop consists of only constant complexity operations
  3. Hence, the asymptotic complexity of Floyd Warshall algorithm is O(n³)

Advantages


It is extremely simple and easy to implement.The Floyd-Warshall algorithm helps in determining all the shortest paths in a graph with positive or negative weight edges. It is known for its simplicity.
