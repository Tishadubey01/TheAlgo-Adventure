Insertion Sort

Examples


Code Example: (in C)


  // Insertion Sort in C
#include <stdio.h>
void insertionSort(int arr[ ], int n)
{
 int i,j,small;
 for (i=0; i<n-1;i++)
 {
  small=arr[i];
  j=i-1;
  while(j>=0 && small<=arr[j])
  {
   arr[j+1]=arr[j];
   j=j-1;
  }
  arr[j+1]=small;
 }
}
int main()
{
 int a[ ] = { 12,31,25,8,32,17 };
 int n = sizeof(a)/sizeof(a[0]);
 int i;
 insertionSort(a,n);
 for(i=0;i<n;i++)
  printf("Elements after sorting: %d",a[i]);
  printf("List before sorting\n");
}

Output:


Elements after sorting: 8 12 17 25 31 32

Advantages:


  1. Simple implementation.
  2. Efficient for small data sets.
  3. It outperforms the Bubble Sort and Selection Sort algorithms.


Daily life example of Insertion Sort


  1. Example of insertion sort is how tailors arrange shirts in a cupboard, they always keep them in sorted order of size and thus insert new shirts at the right position very quickly by moving other shirts forward to keep the right place for a new shirt.
