Selection Sort

Examples


Code Example: (in C)


  // Selection Sort in C
#include <stdio.h>
void selectionSort(int arr[ ], int n)
{
 int i,j,small;
 for (i=0; i<n-1;i++)
 {
  small=i;
  for (j=i+1;j<n;j++)
   if(arr[j]<arr[small])
   {
    small=j;
   }
  int temp=arr[small];
  arr[small]=arr[i];
  arr[i]=temp;
 }
}
int main()
{
 int a[ ] = { 12,31,25,8,32,17 };
 int n = sizeof(a)/sizeof(a[0]);
 int i;
 selectionSort(a,n);
 for(i=0;i<n;i++)
  printf("Elements after sorting: %d",a[i]);
  printf("List before sorting\n");
}

Output:


Elements after sorting: 8 12 17 25 31 32

Advantages:


  1. It performs very well on small lists.
  2. It is an in-place algorithm. It does not require a lot of space for sorting. Only one extra space is required for holding the temporal variable.
  3. Selection sort can be good at checking if everything is already sorted. It is also good to use when memory space is limited.


Daily life example of Selection Sort


  1. If the teacher wants to sort the students based on height.
    Let's say there are students with different heights.
    With this sort, you go through each student and find the tallest student and put him/her at the end. Then you go through the list and find the second tallest student and put him/her before the tallest student and repeat this process until everyone is sorted.

  2. Consider you have 8 shoes pairs with sizes 6 to 13. You find them not in order. Their arrangement from smallest to largest will be an example of selection sort.
