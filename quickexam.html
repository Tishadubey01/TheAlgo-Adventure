#include <stdio.h>
/* function that consider last element as pivot, place the pivot at its exact position, and place smaller elements to left of pivot and greater elements to right of pivot. */
int partition(int a[ ],int start, int end)
{
 int pivot=a[end];
 int i=(start-1);

 for (int j=start;j<=end;j++)
 {
  if (a[j]<pivot)
  {
   i++;
   int t=a[i];
   a[i]=a[j];
   a[j]=t;
  }
 }
 int t=a[i+1];
 a[i+1]=a[end];
 a[end]=t;
 return (i+1);
}

Advantages:


  1. Space required by quick sort is very less, only O(n*log n) additional space is required.
  2. It is in-place since it uses only a small auxiliary stack.


Real-time application of Quicksort


  1. Quicksort is a cache-friendly algorithm as it has a good locality of reference when used for arrays.
  2. It is an in-place sort that does not require any extra storage memory.
  3. Commercial Computing is used in various government and private organizations for the purpose of sorting various data like sorting files by name/date/price, sorting of students by their roll no., sorting of account profile by given id, etc.
